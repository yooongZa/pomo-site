# pomo-site

GitHub Pages 로 서빙된다: https://yooongza.github.io/pomo-site/

- `index.html` — 랜딩
- `privacy/` — 개인정보 처리방침 (App Store Connect 의 "개인정보 처리방침 URL")
- `support/` — 지원/FAQ (App Store 의 "지원 URL")
