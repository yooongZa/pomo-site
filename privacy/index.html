개인정보 처리방침

시행일: 2026년 6월 26일

POMO(이하 "앱")는 계정 없이 사용하는 핸드드립 가이드 앱입니다. 요약하면 — 원두·레시피 등 사용자가 만든 데이터는 기기에 저장됩니다. AI 레시피·원두 인식 기능을 사용할 때에만 해당 요청이 개발자의 Cloudflare(미국) 프록시를 거쳐 Google Gemini로 전달되며, 이때 익명 사용량 메타데이터가 무료 쿼터 관리·비용 모니터링을 위해 보관됩니다. 광고·크로스앱 추적은 없고, 이름·이메일 등 개인 식별정보는 수집하지 않습니다.

1. 기기에 저장되는 정보

  • 원두, 드리퍼, 레시피, 시음 기록, 원두 특징 메모, 앱 설정
  • 원두 패키지 사진(촬영·선택한 경우)

위 정보는 기본적으로 사용자의 기기 안에 저장됩니다. 개발자는 이 목록 자체를 서버로 업로드하거나 동기화하지 않으며, 계정 가입도 없습니다. 다만 AI 기능을 사용하면 그 요청에 포함된 일부 정보가 외부로 전송됩니다(아래 2항 참고).

2. AI 기능 사용 시 외부로 전송·처리되는 정보

AI 레시피 생성·개선과 패키지 사진의 원두 이름 인식(OCR)·추천 기능을 사용할 때에만 아래 처리가 일어납니다. AI 기능을 사용하지 않으면 어떤 정보도 기기를 떠나지 않습니다.

2-1. 전송 경로 — Cloudflare 프록시 경유

이 요청들은 개발자가 운영하는 Cloudflare Workers 프록시(미국 리전)를 거쳐 Google LLC의 Gemini API로 전달됩니다. 프록시는 API 키 보호와 무료 쿼터 관리를 위한 것으로, 요청 본문과 사진을 보관하지 않고 Gemini로 중계한 뒤 응답을 그대로 돌려줍니다. Cloudflare, Inc.는 이 처리에서 개발자의 데이터 처리 위탁자(processor)입니다.

2-2. Google Gemini로 전달되는 내용

  • 원두 이름, 직접 입력한 원두 특징(로스팅·특징 메모)
  • 패키지 사진(사진으로 인식·추천을 요청한 경우) — OCR 및 레시피 추천 목적
  • 레시피 선호(도구·맛 방향·농도)와 시음 평가(레시피 개선을 요청한 경우)

이 과정에서 원두 정보 확인을 위해 웹 검색(Google Search grounding)이 함께 사용될 수 있습니다. 전송된 정보의 처리에는 Google Gemini API 약관이 적용됩니다.

2-3. 프록시가 기록·보관하는 사용량 메타데이터

월 무료 쿼터 집행과 비용 모니터링을 위해, AI 호출 1건마다 아래 메타데이터가 Cloudflare 데이터베이스(미국)에 기록·보관됩니다.

  • 익명 설치 식별자(install_id) — 앱이 무작위로 생성한 안정적 식별자로, 이름·이메일·계정 등 개인 신원과 연결되지 않습니다
  • 호출 시각, 사용 모델명, 요청 상태(성공/실패), 기간(월)
  • 입력·출력 토큰 수, 추정 비용, 응답 지연시간

이 메타데이터에는 레시피 내용이나 사진이 포함되지 않으며, 개인 식별정보도 들어 있지 않습니다.

3. Apple Watch 동기화

원두·레시피는 Apple의 WatchConnectivity 프레임워크를 통해 사용자의 iPhone과 Apple Watch 사이에서만 직접 전달됩니다. 외부 서버를 거치지 않습니다.

4. 수집하지 않는 것

  • 이름, 연락처, 위치 등 개인 식별 정보
  • 광고 식별자(IDFA), 광고·추적 SDK
  • 제3자 분석(애널리틱스) SDK

익명 install_id는 위 2-3항의 사용량 관리에만 쓰이며, 다른 회사의 앱·웹사이트에 걸친 추적(크로스앱 트래킹)에는 사용되지 않습니다.

5. 데이터 삭제

앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다. AI 사용량 메타데이터(2-3항)는 익명 식별자 기준으로 보관되며 개인 신원과 연결되지 않습니다. 해당 메타데이터의 삭제를 원하시면 install_id와 함께 아래 문의처로 요청해 주세요.

6. 변경 사항

English summary

POMO stores the data you create (beans, recipes, tasting logs, photos) on your device; the developer does not upload or sync this catalog, and there is no account. When you use the AI recipe or bean-recognition features, those requests are relayed through the developer's Cloudflare Workers proxy (US region) to Google's Gemini API. The proxy adds the API key and forwards your request — including the bean name, brewing preferences, tasting feedback, and any package photo (for OCR and recipe recommendations, with Google Search grounding) — without storing the request body or the photo, and returns Gemini's response. For each AI call, anonymous usage metadata is logged to a Cloudflare database for monthly free-quota enforcement and cost monitoring: an anonymous install identifier (not linked to your name, email, or any account), timestamp, model name, request status, input/output token counts, estimated cost, and latency. This metadata contains no recipe content, no photos, and no personal identifiers. Cloudflare, Inc. acts as a data processor; Gemini processing is governed by the Gemini API terms. No ads, no third-party analytics, and no cross-app tracking. Deleting the app deletes all on-device data. Contact: sukmack@gmail.com
